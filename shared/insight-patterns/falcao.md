# Falcão — Insight Patterns to capture

Call wiki-remember.sh when you observe any of these:

1. **Post flop (<50% of 7-day baseline engagement within 2h of posting)**
   Title: `Falcão post flop — [platform] [hook-type]`
   Content: post URL/ID, platform, expected baseline, actual engagement at 2h, hypothesis for why it flopped, whether the hook / thumbnail / timing was the miss.

2. **New high-engagement hook (>2x baseline within 2h)**
   Title: `Falcão breakout hook — [specific hook structure]`
   Content: exact hook text, platform, engagement numbers, timing, whether replicable (vs. lucky viral moment). If replicable, propose a template.

3. **Platform algorithm shift (category-wide over 30% swing vs. prior 30-day trend)**
   Title: `Platform algo shift — [platform] favors [category]`
   Content: platform, category (carousel/video/text), direction, affected posts with data, hypothesis about algo change.

4. **Tool regression (publishing tool silently failing or behaving unexpectedly)**
   Title: `[tool] regression — [specific symptom]`
   Content: tool name, exact symptom, retry behavior, workaround if any, date noticed.

5. **Credential/API expiry pattern**
   Title: `[platform] token expiry — [interval observed]`
   Content: when token expired, symptom before error surfaced, detection lag, refresh procedure.

DO NOT write for routine successful posts, heartbeats, or standard publishing completion.
